Academic Projects

LOVE2041

This individual project used Perl and CGI to create a website which mimicked the functionality of popular dating sites.

Bleeding Edge Flight Scheduling System

A Java based web servlet which gives users the 'best' possible flight plans from one airport to another using a given dataset. Factors for determining the best flight plan could include price, travel distance, time etc.

GlobeTrotter

A web based application written using the Django framework that would show users popular restaurants and attractions within a given city. This was done using REST APIs from Google, Yelp, and Yahoo.
